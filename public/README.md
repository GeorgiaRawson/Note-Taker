# Note Taker
           
           ## Description

           A full stack note taker application which allows users to: Create new notes, Read existing notes, Update notes, Delete notes. The application uses Node and Express.
            
           ## Installation 

           Git Clone to local machine. Initalise and install Node and Express from terminal.  
   
           ## Useage 
           
           Tasks can be added to the application in browser, and then edited or deleted with use of the buttons.
